## Escenario elegido
Análisis de ventas de una pequeña empresa.

## Descripción del proyecto
En este trabajo se realizó un análisis básico de ventas utilizando Python.

La idea del proyecto fue trabajar con un archivo CSV que contiene datos de ventas y, a partir de eso, obtener información útil como:
- ventas totales,
- producto más vendido,
- ventas por mes,
- y un gráfico para visualizar mejor los resultados.

También se trabajó con Git y GitHub para registrar cambios y organizar el proyecto de manera similar a un entorno de trabajo real.
